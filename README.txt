The game is played through running Main.py with the library Pillow and Tkinter required with Python 3.7
Chess, a 8x8 board with 2 sets of pieces one for each player.
The pieces include a King, Queen, Rook, Bishop, Knight, and Pawn.
Move the piece throughout the board to put the opponent in check / checkmate.
